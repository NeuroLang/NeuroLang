This directory contains the following files :

- performed_modifications.txt :
  This file details the error and collision messages encountered when using the
 "parser='lalr'"option in the Lark constructor and the solutions applied to remove them.

- grammar_lark.png and grammar_lark.svg :
  These are the graphical representation of the NeuroLang grammar written in Lark
  format in ../standard_syntax.py file.

In both of the images, the dashed arrows represent the removed relationships,
the red arrows and expressions represent the new ones and the modifications.

- grammar_lark.txt :
  This is the grammar in PlantUML format to generate the preceding figures with
  https://www.planttext.com/
